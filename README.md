# Me Fashion

An e-commerce page with shopping-cart capabilities

[**Live Demo**](https://ipeng5.github.io/me-fashion/)

## Getting started

```
git clone https://github.com/ipeng5/me-fashion.git
cd memory-card
npm install
npm start
```

## Built with

- React
- Sass
- Fake Store API
